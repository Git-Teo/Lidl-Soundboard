boolean org.schabi.newpipe.player.BasePlayer.isApproachingPlaybackEdge ( final long  timeToEndMillis)
inline

Called to check if the currently playing stream is approaching the end of its playback. Implementation should return true when the current playback position is progressing within timeToEndMillis or less to its playback during.

May be called at any time.

Implements org.schabi.newpipe.player.playback.PlaybackListener.
